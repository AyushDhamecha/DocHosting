Step 3: App.config Parameters configuration

Update key parameters under the <appSettings> schema in App.config:

Configuration Key Purpose Value Configuration Instance
ApplicationUri Domain address of target environment where tests are triggered <add key="ApplicationUri" value="https://opcoreauto1.icertis.com" />
ExecuteTestCases Comma-separated execution values targeting specific unique IDs <add key="ExecuteTestCases" value="75668" />

6

Run Isolated Suite Method

Execute

Navigate directly to the test signature inside the declared C# file. **Right-click** on the test method name and click on **Run Tests**.

  • Compiling & Build Phase: Visual Studio validates dependent code schemas and generates localized binaries.
  • Browser Initialization: ChromeDriver initiates, launching a specialized Google Chrome context pointing to your configured environment.
  • Authentication Automation: The script injects configuration parameters and cookies to bypass typical login flows.
  • Asserting Test Parameters: The browser executes simulated interactions based on XML dataset properties, recording passes and assertion statuses automatically.
